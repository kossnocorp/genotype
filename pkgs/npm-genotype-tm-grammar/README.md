# Genotype TextMate Grammar

Genotype Programming Language TextMate grammar package. It is used for syntax highlighting in [Visual Studio Code](https://code.visualstudio.com/), [Shiki](https://shiki.style/), etc.

To get syntax highlighting for Genotype in Visual Studio Code, install the [Genotype extension](https://marketplace.visualstudio.com/items?itemName=nocorp.genotype).

See the [Genotype website](https://genotype-lang.org) for more information about the project.
