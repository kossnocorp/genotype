# Litty Macro Tests

Literally adds literals to Rust enum variants, struct fields, and unit structs.

`litty_macro_tests` holds tests for the `litty` crate's procedural macros and should not be used directly. See [the `litty` crate](https://crates.io/crates/litty) for more info.
